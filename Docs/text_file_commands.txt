 
User Variables

      %NODE%		Current Telnet Node
      %TIMEOFDAY%  	Morning, Afternoon or Evening
      %USERNAME%  	Current User's Name
      %U_LDATE%  	Current User's Last On Date
      %U_LTIME%  	Current User's Last On Time
      %U_LEVEL%  	Current User's Level
      %U_LOGONS% 	Current User's Number of Logons
      %U_POSTS% 	Current User's Total Messages Posted (not including email)
      %U_RATIO%  	Current User's Post / Logon Ratio (not including email)     
      %U_ADDR%  	Current User's Address
      %U_CITYSTATE%  	Current User's City and State
      %U_ALIAS%  	Current User's Chat Alias
      %IP%  		Current User's IP Address
